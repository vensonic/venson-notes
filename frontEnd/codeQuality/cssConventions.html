# CSS规范

# css属性书写顺序

  1. 布局定位属性：display / position / float / clear / visibility / overflow
  2. 自身属性：width / height / margin / padding / border / background
  3. 文本属性：color / font / text-decoration / text-align / vertical-align / white- space / break-word
  4. 其他属性（CSS3）：content / cursor / border-radius / box-shadow / text-shadow / background:linear-gradient …
.class-name {
    display: block;
    position: relative;
    float: left;
    width: 100px;
    height: 100px;
    margin: 0 10px;
    padding: 20px 0;
    font-family: Arial, 'Helvetica Neue', Helvetica, sans-serif;
    color: #333;
    background: rgba(0,0,0,.5);
    -webkit-border-radius: 10px;
    -moz-border-radius: 10px;
    -o-border-radius: 10px;
    -ms-border-radius: 10px;
    border-radius: 10px; 
}

# 选择器

  • 尽量少用通用选择器（*）
  • 避免使用标签选择器
  • 避免使用ID选择器

# 使用缩写属性

/* good */
.class-name {
  margin: 10px 20px 30px 40px;
  /* or */
  margin: 10px 20px;
  /* or */
  margin: 10px 0 20px;
}

/* bad */
.class-name {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
}

# 数值与单位

去掉小数点的0

/* good */
.class-name {
  opacity: .8;
  color: rgba(255, 255, 255, .5);
  top: -.5px;
}

/* bad */
.class-name {
  opacity: 0.8;
  color: rgba(255, 255, 255, 0.5);
  top: -0.5px;
}

属性值为0的可省略单位

/* good */
.class-name {
  margin: 0 auto;
}

/* bad */
.class-name {
  margin: 0px auto;
}

十六进制值使用小写和简写

/* good */
.class-name {
  color: #fc0;
}

/* bad */
.class-name {
  color: #ffcc00;
}

# 代码风格

使用less或sass时类名嵌套最好不超过3层

// good
.main {
    
  .wrap {
      
  }
}

.list {

  .item {

  }
}

// bad
.main {
  
  .wrap {
    
    .list {
      
      .item {

      }
    }
  }
}
